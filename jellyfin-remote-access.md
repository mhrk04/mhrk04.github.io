# How to remote access jellyfin
1) enable remote connection on jellyfin website
2) windows firewall -> inbound rules -> rule type:TCP -> port:80,443
3) install JF-secure from https://github.com/dab2020/JF-secure and install
4) edit file domain name on Caddyfile and copy to C:location on caddy bin
5) click on initial and and select select-me-in-NSSM.bat
